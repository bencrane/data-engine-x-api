## Key Comparisons: Manufacturing vs IT

| Dimension | Manufacturing | IT/Prof Services |
|---|---|---|
| Unique companies | 12,370 | 12,837 |
| Total transaction rows | 510,275 | 57,948 |
| Avg transactions per company | 41 | 5 |
| First-time awardee % | 39% | 49% |
| Avg award per transaction | $14,988 | $177,544 |
| Median award per transaction | $371 | $0 |
| POC name coverage | 98% | 98% |
| Phone coverage | 99% | 100% |
| Website coverage | 76% | 81% |
| Top state | CA (1,484) | VA (3,195) |
| Geographic concentration | Spread nationally | 41% in DC corridor |

**Manufacturing** is a high-volume, low-dollar game — lots of purchase orders for parts, supplies, and food. Repeat awardees dominate (61% repeat vs 39% first-time).

**IT/Professional Services** is a low-volume, high-dollar game — consulting, R&D, and systems integration contracts. More even split between first-time and repeat (49/51). Heavily concentrated in the DC area.

For outbound, both verticals have near-universal contact data quality. Manufacturing has slightly more phone numbers; IT has slightly more websites. Both are outbound-ready.
